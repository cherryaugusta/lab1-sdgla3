Top project AI Workflow Audit-ready

Consumer Duty Evidence Engine

Django · DRF · React · TypeScript · PostgreSQL · Redis · Celery · Channels · pgvector

AI-assisted evidence review workflow for regulated financial-services artefacts. The system ingests complaints, disclosures, transcripts, scripts, and policy materials; extracts structured claims; maps them to Consumer Duty outcome areas; scores evidence sufficiency; detects contradictions; routes uncertain cases into analyst review; and produces audit-ready outputs with traceable citations, evaluation metrics, and observable workflow state.

  • Project link: GitHub repository
  • Async ingestion and parsing pipeline with Celery, Redis, and WebSocket updates
  • Evidence sufficiency scoring across supported, weak, missing, contradictory, and stale evidence states
  • Human review queues with assignment, approval, escalation, override controls, and audit logging
  • Evaluation harness with regression-style metrics, citation validation, and degraded-mode safeguards
Top project Rules-first AI Version-aware

Agentic Compliance Auditor

Django · DRF · React · TypeScript · PostgreSQL · Redis · Celery · Channels · pgvector

Rules-first, AI-assisted policy-control audit workflow for detecting contradictions, drift, and version misalignment across internal policies and external guidance. It ingests policy material, parses sections, extracts normalised control statements, compares versions and source pairs, generates typed findings with citations, routes findings into review tasks, preserves audit lineage, and exposes evaluation and observability outputs through API and UI surfaces.

  • Project link: GitHub repository
  • Version-aware document ingestion, lineage tracking, section parsing, and statement normalisation
  • Rules-first contradiction and drift detection with typed findings and cited evidence
  • Review-task routing, audit events, replayable runs, and exportable finding packets
  • Evaluation reporting and degraded-mode visibility when prompt or provider support is unavailable

  • Audit timeline, operational metrics, and platform-oriented AI lifecycle modelling
